Detailed Description

Structure representing a Layer.

Layers are containers for graphics, event handlers, information etc. They can be attached (i.e. pushed onto a LayerStack), which will let them display graphics or catch events. If a layer receives an Event it can decide to handle it, or let it pass through to the layer below
